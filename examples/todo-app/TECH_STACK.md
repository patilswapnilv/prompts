# Technology Stack — Todo App Example

**Template version:** 1.0  
**Example:** Minimal stack for client-only todo MVP.

## 1. Stack Overview
- **Type**: Single-page app (SPA), client-only for this example.
- **Pattern**: Vanilla JS or minimal framework (e.g. React/Vue) + localStorage.
- **Deployment**: Static hosting (e.g. GitHub Pages, Netlify).

## 2. Frontend Stack
- **Runtime**: Browser (ES2020+).
- **Optional framework**: React 18 or Vue 3, or vanilla JS.
- **Styling**: CSS (or Tailwind/CSS modules if using a framework).
- **State**: In-memory list + localStorage for persistence.

## 3. Backend Stack
- **None** for this example (see PRD §7 OUT OF SCOPE). Data in localStorage only.

## 4. Development
- **Node**: v18+ for tooling (if any).
- **Package manager**: npm or pnpm.
- **Scripts**: `build`, `dev` (if using a bundler), `lint`.

## 5. Environment
- No env vars required for client-only example.

## 6. References
- PRD §7 (OUT OF SCOPE). BACKEND_STRUCTURE omitted for this example; add if you introduce an API later.
